About Me

Learning. Getting stuck, being persistent...and learning. That is the thing that has always driven me, whether as a writer, a developer, a coach or any of the other "things" I've been and done up until now.

I'm excited about building things. I love to explore. I'm looking for a great team to build great things with and learn a ton from. I love creating things with the MERN stack, but I'm also looking forward to adding new languages and skills to my tool chest.

I live near Portland, in the beautiful Pacific Northwest, but I've worked either fully or partially remote for many years. I couldn't be more excited about this next step in my career as a software developer.

Projects

SharedBrew

  • Social Recipe Book / Shopping List app for the home brewing community.
  • Users can create and share recipes, then create shopping lists from them.
  • Initially using Firebase, this project was refactored into a full MEAN stack.

ReallySmallStories

  • Very short stories and pieces of literature based on Unsplash images chosen by the user.
  • Users can create and share stories, then read and upvote others.
  • Uses Node, Express, Handlebars, MongoDB, Travis CI, and hosted on Heroku.

Hockey Playground

  • Hockey quiz / NHL video app pulling data from YouTube API.
  • Designed primarily to extract time from the lives of hockey fans.
  • Built primarily with jQuery, using Bourbon / Neat / Bitters for layout.

Pinstruments

  • Pinboard style app for musicians and fans to oggle musical instruments.
  • Built with Ruby on Rails / PostgreSQL, and using Bootstrap on the front end.
  • Utilizes AWS S3 storage for the images.

Danish Bench

Not a virtual bench

  • I refactored donated mahogany lumber for this Danish / Japanese style bench.
  • Danish cording is used throughout the application to enable users to "sit down".
